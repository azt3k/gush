Gush
====

Provides touch scroll support for android < 3 devices as a work around for lack of overflow support.


Options / Defaults
------------------

````json
{
    x: true,
    y: true
}
````

- `x` : enables / disables scroll on x axis
- `y` : enables / disables scroll on y axis


Limitations
-----------

- only supports android < 3 devices
